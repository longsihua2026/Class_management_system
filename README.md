# 前言
> 该项目是2023-2024综合课程设计的作业
该项目名称为班级信息管理系统。其中运用c++和数据结构的知识进行构建和完成系统。

# 班级信息管理系统功能
1. c++类的操作的使用
2. 多文件操作的使用
3. 数据的文件存储
4. 中间件编译运行
5. 链式存储链表的使用（基本操作）
